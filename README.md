# Omero - Indie Games Studio

A clone of the Omero Indie Games Studio WordPress theme. Built with vanilla HTML, CSS, and JavaScript.

## Features

- **Hero Section** - Full-width cinematic banner with overlay and particle effect
- **Featured Games** - Game cards with hover effects and trailer play buttons
- **Countdown Timer** - For upcoming game launches
- **About Section** - Studio intro with stats
- **Team Section** - Team member cards with social links
- **News/Blog** - Devlog updates
- **Newsletter** - Email signup form
- **Responsive** - Mobile-friendly design

## How to Run

Open `index.html` in a web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling
- `script.js` - Countdown, navigation, smooth scroll
